# Bluetooth
##環境

Swift 5.4.2
Xcode12.5.1

##機能

Bluetoothを使って足裏のデバイスから圧力値を受け取り表示する．
https://drive.google.com/file/d/1SGg1N0KJWNfHE8NsMwvXiF_v6DZqrZkg/view?usp=sharing

##注意点

Bluetoothを使うためエミュレータでの実行はできず，実機で行う必要がある．
また，圧力検知にArduinoを用いているためそちらの用意も必要である．
